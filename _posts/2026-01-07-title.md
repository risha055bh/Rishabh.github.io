## üìò Overview

### What is a Logic App?
Azure Logic Apps is a cloud-based service by Microsoft Azure that enables users to create and run automated workflows integrating various applications, systems, and services.

### What is a Requests Trigger?
In Azure Logic Apps, the "Requests" trigger initiates a workflow whenever an HTTP request is received. It‚Äôs commonly used to expose endpoints for automation.

### What is IDOR?
IDOR (Insecure Direct Object Reference) is a vulnerability where an application exposes internal objects based on user-supplied input. Attackers can manipulate parameters (e.g., in URLs or forms) to access unauthorized data.

---

## üß† Hint
- IDORs are common in web applications.
- HTTP requests support multiple methods (GET, POST, etc.).
